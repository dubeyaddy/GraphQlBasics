# GraphQL Basics 🚀

This repository contains **8 example files** demonstrating the fundamentals of GraphQL in Node.js. It covers everything from setting up a simple GraphQL server to integrating with **Express** and **Apollo Server**, as well as performing basic **CRUD operations** with GraphQL.

---

## 📂 Project Structure
- **Basic GraphQL server** – A minimal GraphQL setup using `apollo-server`.
- **Express + GraphQL** – Integration of GraphQL with an Express server.
- **Apollo Server examples** – Demonstrations of queries, mutations, and schema definitions.
- **CRUD operations** – Examples of creating, reading, updating, and deleting data using GraphQL resolvers.
- **Sample data** – In-memory datasets used for testing queries and mutations.
